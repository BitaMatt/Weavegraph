# MindMapInterface

客户关系网络思维导图工具 - 可视化管理个人和企业的关系网络

## 功能特点

### 🎯 核心功能

- **关系网络可视化**：使用Cytoscape.js实现直观的关系网络图
- **智能搜索**：快速搜索和定位节点
- **关系类型管理**：支持多种关系类型的添加和编辑
- **联系方式集成**：直接启动和聚焦QQ、微信、WhatsApp、Outlook等应用
- **数据管理**：本地存储，支持多个存档的创建和编辑
- **加载动画**：科技感的网络背景效果和倒计时进入按钮

### 📋 关系类型

- **生理关系**：父亲、母亲、配偶、子女、兄弟、姐妹、祖父母、外祖父母、孙子女
- **社会关系**：朋友、同学、同事、邻居、合作伙伴、客户、供应商、竞争对手、导师、学生
- **法律关系**：夫妻、父母、子女、兄弟姐妹、祖父母、外祖父母、孙子女、法定监护人、被监护人、债权人、债务人、雇主、雇员
- **经济关系**：合作伙伴、客户、供应商、竞争对手、投资者、被投资者、债权人、债务人
- **情感关系**：朋友、恋人、配偶、家人、同事、同学
- **其他关系**：邻居、网友、笔友、其他

### 🔧 技术特性

- **跨平台**：支持Windows和macOS
- **响应式设计**：适配不同屏幕尺寸
- **快捷键支持**：Ctrl+S保存，Ctrl+L切换布局，Ctrl+Shift+I开发者工具
- **国际化支持**：多语言界面
- **数据安全**：本地存储，数据不上传

## 快速开始

### 系统要求

- Windows 10/11 或 macOS
- Node.js 18.0+
- npm 9.0+

### 安装步骤

1. 克隆项目：`git clone <repository-url>`
2. 安装依赖：`npm install`
3. 启动开发模式：`npm start`

### 构建步骤

- 构建Windows版本：`npm run build:win`
- 构建macOS版本：`npm run build:mac`
- 构建所有版本：`npm run build`

## 使用指南

### 基本操作

1. **创建节点**：点击"创建新节点"按钮
2. **添加关系**：选择两个节点，点击"添加关系"按钮
3. **编辑节点**：双击节点进行编辑
4. **删除节点**：选择节点后按Delete键
5. **保存数据**：使用Ctrl+S或点击保存按钮
6. **加载存档**：从存档列表中选择
7. **编辑存档**：点击存档旁边的"编辑"按钮

### 联系方式

- **电话**：直接拨打
- **QQ**：启动并聚焦QQ窗口
- **微信**：启动并聚焦微信窗口
- **WhatsApp**：启动并聚焦WhatsApp窗口
- **邮箱**：打开默认邮件客户端
- **地址**：在地图中显示

## 技术栈

- **前端**：HTML5, CSS3, JavaScript
- **框架**：Electron
- **可视化**：Cytoscape.js
- **存储**：localStorage
- **构建**：electron-builder
